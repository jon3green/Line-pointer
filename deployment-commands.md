# Line Pointer - Deployment Commands

## Vercel Commands
```bash
# Login to Vercel
vercel login

# Link project to Vercel
vercel link

# Deploy to production
vercel --prod

# Deploy to preview
vercel

# Check deployment status
vercel ls

# View deployment logs
vercel logs

# Open project in browser
vercel open
```

## GitHub Commands
```bash
# Check status
git status

# Stage all changes
git add .

# Commit changes
git commit -m "Your message"

# Push to GitHub (triggers Vercel auto-deploy)
git push origin main

# View recent commits
git log --oneline -5
```

## Development
```bash
# Run local dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Cursor AI Features
- Press Cmd+K for AI chat
- Press Cmd+L for AI editing
- Use Git panel (left sidebar) for commits
- Built-in terminal at bottom
